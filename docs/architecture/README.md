# Architecture Documentation

This directory contains architecture documentation for the Bharat-Setu project.

## Overview

For the complete architecture overview, see the [Design Document](../specs/bharat-setu/design.md#architecture).

## Key Components

1. **Voice Processing Layer** - Amazon Transcribe and Polly integration
2. **Intelligence Layer** - Amazon Bedrock Agent with RAG
3. **Data Layer** - DynamoDB and S3 storage
4. **API Layer** - API Gateway and Lambda functions
5. **Security Layer** - Guardrails and IAM

## Architecture Diagrams

Architecture diagrams and detailed component documentation will be added as the implementation progresses.
